# Scientific Computing & Mathematics with Python (NumPy & Matplotlib)

This repository contains a comprehensive collection of scripts, laboratory exercises, and advanced projects developed using Python for scientific computing. The focus ranges from fundamental syntax to the implementation of complex mathematical algorithms and data visualization.

## Key Skills
* **Numerical Computing:** Advanced use of `NumPy` for multi-dimensional arrays and vectorized operations.
* **Algorithmic Design:** Re-engineering of matrix algebra, determinant solvers (Laplace expansion), and recursive functions (Fibonacci).
* **Mathematics:** Polynomial calculus (roots, derivatives, division), linear systems (Rouch√©-Frobenius), and 3D geometry.
* **Advanced Visualization:** Multi-panel plots, 3D projections, and statistical charting using `Matplotlib`.
* **Software Engineering:** Input validation, error handling, and modular function documentation (docstrings).

---

## Repository Structure

### 1. Course Foundations (`Theory-Notes/`)
* **`Script_1-4.py`**: A journey through Python basics, logic branching, loops, and convergent sequences.
* **`polinomios.py`**: Comparative study of polynomial library conventions (Increasing vs. Decreasing powers).
* **`Resumen.py` (The Master Cheat Sheet)**: A comprehensive reference for SciPy integration, linear algebra, and advanced plotting aesthetics.

### 2. Applied Practices (`Practices/`)
* **`Practica-1`**: Precision arithmetic, complex number handling (`cmath`), and variable logic.
* **`Practica-2`**: Vectorized operations and computational verification of matrix theorems.
* **`Practica-3`**: Advanced slicing, nested data structures, and multi-panel function analysis.
* **`Practica-4`**: Parametric geometric generators (circles, ellipses) and manual matrix sum/product algorithms.
* **`Practica-5`**: Numerical convergence analysis and iterative safety monitoring for large-scale loops.
* **`Practica-6`**: Advanced polynomial root finding and symbolic differentiation.
* **`Practica-Adicional`**: A capstone of challenges including financial modeling, 3D spatial distance engines, and prime number theory.

### 3. Deliverables & Exam (`Projects/`)
* **`Entregable.py` (Final Project)**:
    * **3D Geometry Engine**: An automated solver for the relative position and distance of lines in $\mathbb{R}^3$.
    * **Calculus Suite**: A master function that calculates roots, extrema, and inflection points of any polynomial.
    * **Input Validation**: Interactive logic to detect perfect powers with robust error handling.
* **`Examen.py` (Final Exam)**:
    * Real-time implementation of a cube root approximation algorithm.
    * Multi-frequency signal visualization and comparison.
    * Polynomial divisibility testing and algebraic reconstruction.
