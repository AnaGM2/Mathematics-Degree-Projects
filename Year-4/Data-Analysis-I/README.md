# Data Analysis I

This repository contains the practical implementation of multivariate statistical techniques in R. The course focuses on dimensionality reduction, pattern recognition, and predictive modeling, providing the mathematical foundation for modern Machine Learning algorithms.

## Repository Structure
The content is organized as follows:
* `Practices/`: Scripts from 8 thematic chapters, implementing algorithms for data transformation and analysis.
* `Solved-Exams/`: Solutions to past exams used as training, demonstrating the ability to apply theoretical concepts to complex problems.

---

## Chapters and topics covered

* Chapter 1: Introduction to Data Analysis
* Chapter 2: Principal Component Analysis (PCA)
* Chapter 3: Correspondence Analysis
* Chapter 4: Multidimensional Scaling (MDS)
* Chapter 5: Cluster Analysis
* Chapter 6: Linear Regression
* Chapter 7: Discriminant Analysis
* Chapter 8: Analysis of Variance (ANOVA)

## Connection to AI & Machine Learning
* **Unsupervised Learning:** Chapters 2 and 5 are the basis for feature engineering and data segmentation in AI.
* **Dimensionality Reduction:** Techniques like PCA are essential when dealing with high-dimensional datasets.
* **Statistical Rigor:** Understanding ANOVA and Linear Regression ensures that the models developed are not just black boxes.
